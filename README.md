# NAPPN Conference Proceedings Templates and Submission Instructions

This repository contains the LaTex and Word template files for conference proceedings manuscripts submitted to the North American Plant Phenotyping Network conferences.

The templates are derived from the SPIE Proceedings template available through Overleaf and licenced under a Creative Commons Attribution 4.0 International (CC-BY-4.0) licence: https://www.overleaf.com/latex/templates/spie-proceedings-style-template-and-guidelines-for-authors/qpkhfttzvnhz.

## Abstract template and guidelines

For abstracts, please use the [Word template](https://github.com/nfahlgren/nappn-proceedings-template/blob/main/word-templates/2023-NAPPN-abstract-template.docx). Abstracts should be no longer than 250 words, 12pt Font, and 1" margins.

## Proceedings paper templates and guidelines

For proceedings papers, please use either the [Word template](https://github.com/nfahlgren/nappn-proceedings-template/blob/main/word-templates/2023-NAPPN-conference-proceedings-template.docx) or the LaTex template (this repository). Conference proceedings papers should be no longer than 4 pages (excluding references).


## Submission instructions

Detailed submission instructions are available at https://www.plantphenotyping.org/conference/research-submissions
